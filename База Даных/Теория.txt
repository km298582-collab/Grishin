let и const имеют блочную область видмости и видны только в блоке кода в котором они объявлены:

    console.log('NORMAL');
    alert('NORMAL.');
}

{
    console.log('BAD');
    alert('BAD...');
}

- 4 модуль - 
База данных - это структурированное хранилище информации. информация может быть представлена разными типами данных. строгая типизация. 

есть два основных вида баз данных: 
1) no sql базы данных - не реляционные (пример: mongo DB.)
2) реляционные базы данных - базы данных представленныые в табличном виде (пример: oracl, mySQL, postgresSQL, SQLlite, SQLserver)

основными элементами реляционных баз данных являются таблицы, атрибуты, связи

таблицы (сущности, модели данных) - в таблицах есть колонки, первая строка является заголовоком (атрибутом) колонки

типы даных postgresSQL -
для поля name - VARCHAR
для неограниченного текста - text 
timestamp - дата и время
date - дата без времени
integer - целые числа
decimal или numeric - любые другие
serial - для уникальных полей. поле id является первичным ключои 

- ограничения - 
NOT NULL - нужно обязательное значение внутри ячейки иначе запись не создасться 
defaultValue = 0 = "закрыт" - автоматически в случае отсутствия значения даёт базовое значение 
enum - огранчиение 
now() - в случае отсутствия наполенния берёт актуальную дату врем

Атомарность атрибтов неделимость а также единичные жкзамепляры значений атрибутов
уникальный атбрибут в кажо таблице авто 
1 нрмальная форсма Каждая ячейка содержит только одно значение. -
Нет повторяющихся групп или списков.Строки уникальны (есть первичный ключ).
2. НФ - База уже находится в 1NF.Все неключевые поля зависят от всего составного ключа, а не от его части.
3.Суть: Устранение транзитивных зависимостей.База уже находится в 2NF. -
Неключевые поля не должны зависеть от других неключевых полей (зависимость только от ключа).
